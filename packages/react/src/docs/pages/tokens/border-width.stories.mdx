# Border Widths

Border width tokens are variables in a design system that represent the thickness of borders. By using these tokens instead of direct pixel values, consistency is maintained and updates become simpler, as changing a token's value reflects everywhere it's used.

## Table of Contents

-   [All Border Widths](#all-border-widths)

## All Border Widths

Border width tokens ensure that the thickness of borders is consistent across your application. They are named semantically to describe their purpose, such as "thin" or "thick".
